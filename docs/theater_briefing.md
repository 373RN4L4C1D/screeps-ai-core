# OPERATION GRIDIRON - Theater Briefing

## Mission Statement
Establish autonomous theater operations in adverse terrain conditions
using Naval Mobile Construction Battalion protocols.

## Unit Designations

### combat_logistics (CLC)
- **Mission:** Resource extraction, supply chain security
- **Loadout:** WORK, CARRY, MOVE-heavy
- **Tactics:** Sticky assignment to supply points

### seabee (SEEBEE)  
- **Mission:** Route clearance, FOB construction, MSR maintenance
- **Loadout:** Balanced for construction efficiency
- **Tactics:** Priority-based construction matrix

### area_command (ACC)
- **Mission:** Theater security level advancement
- **Loadout:** WORK-focused with mobility
- **Tactics:** Forward logistics acquisition from containers

## Chain of Command
1. Theater Command (main.js)
2. Corps of Engineers (infrastructure)
3. Unit Commanders (individual roles)

## Operational Notes
- Swamp terrain requires MOVE-heavy loadouts
- Roads are force multipliers in adverse terrain
- Construction Sites enable unit deployment
- Cache invalidation prevents intelligence stagnation
